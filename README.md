# human-cortical-organoid-signal-analysis
Human cortical organoid/spheroid signal analysis

currently working data conversion from RHS to NWB to NPZ for human cortical spheroid data, and working on phase space metric and symbolic dynamics metric for testing of whether the organoid/spheroid is learning
